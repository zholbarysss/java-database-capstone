# User Stories

## Doctor
- As a doctor, I want to view patient records.
- As a doctor, I want to update patient diagnoses.
- As a doctor, I want to prescribe medications.

## Patient
- As a patient, I want to register an account.
- As a patient, I want to book an appointment.
- As a patient, I want to view my medical history.

## Admin
- As an admin, I want to manage users.
- As an admin, I want to manage doctors.
- As an admin, I want to manage appointments.
